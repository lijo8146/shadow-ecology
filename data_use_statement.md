# Data Use Statement — Tribal-Sourced Data

**Status:** DRAFT to be reviewed and finalized in consultation with Tribal partners before any Tribal-sourced data is incorporated

## Purpose

Shadow Ecology draws on monitoring network context and, in Phase 3, collaborative input regarding monitoring gaps in Tribal contexts. This statement sets out how that data and context is handled, distinct from the project's federally-sourced open datasets (USGS, NOAA, NLCD, Sentinel-2, Landsat), which are used under their standard public-domain/open-data terms and are not covered by this document.

## Principles

- Tribal-sourced monitoring data, contextual information, or collaborative input is used only with the agreement of the relevant Tribal partner(s), on terms set by that partner.
- Such data is **not** committed to the public GitHub repository and is **not** redistributed beyond the scope agreed with the partner.
- Where Tribal-sourced input is used to characterize monitoring gaps or inform driver attribution (Phase 3), the partner's preferred level of attribution, aggregation, or anonymization is followed.
- Results from Phase 3 involving Tribal lands are shared back with the relevant Tribal environmental programs, per the project workplan, before or alongside any external publication.
- This project does not claim ownership over Tribal environmental knowledge or monitoring data; it uses such data, where shared, strictly for the stated research purpose and reporting terms agreed with each partner.

## Scope of This Document

This is a placeholder/draft to be finalized once specific Tribal partner agreements for Phase 3 (monitoring gap characterization) are in place. It should be reviewed against each partner's own data governance requirements rather than treated as a substitute for a formal data sharing agreement.
